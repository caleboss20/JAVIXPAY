# javixpay

Fast, secure cross-border payments for Africa. A modern mobile app enabling seamless
intra-African transactions with low fees and real-time exchange rates.

## Overview

javixpay is a fintech solution designed to simplify cross-border payments across African nations.
Our mission is to make international transactions fast, affordable, and accessible to everyone.

**Technology Stack:**
- Mobile: React Native
- Backend: Golang
- Status: Active Development

## Features

- Fast Transactions - Send money across borders in minutes
- Secure - Bank-grade encryption and compliance
- Real-time Rates - Live exchange rates with no hidden fees
- Low Fees - Competitive pricing for intra-African transfers
- Mobile-First - Intuitive React Native interface
- Scalable - Golang backend built for growth

## Getting Started

### Prerequisites

- Node.js (v16+)
- React Native CLI
- Go (v1.18+)
- Git

### Mobile App Setup

```bash
# Clone repository
git clone https://github.com/caleboss20/javixpay.git
cd javixpay/mobile

# Install dependencies
npm install

# Start development server
npm start

# Run on Android
npm run android

# Run on iOS
npm run ios
```

### Backend Setup

```bash
cd javixpay/backend

# Install dependencies
go mod download

# Run server
go run main.go
```
